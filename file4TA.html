Poker Game

Question 1


To deal Press "Deal" button. The deck will be split among 4 players by default.

Question 2


To Multi-deal, Enter the number of deals in the prompt box that appear upon clicking multi-deal. Note: The program might take a while depending on the number of deals input!!

Question 3


You can Alter the power of a card using the change power. "Only a single input must be passed at a time."

Question 4


Set power groups lets you to assign a specific power to a group of cards, if a player comes in possession of the specified group they will be awarded a bonus.

Question 5


This button helps you set a custom number of players and a provide the players with a custom number of cards.
